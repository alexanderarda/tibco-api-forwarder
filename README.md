# Tibco Mashery API Forwarder use Lumen PHP Framework


When use Tibco Mashery API Management, if you use trial version you only can use several domains only such as heroku and domain. Then you have to create forwarder to your own domain from heroku/azure app to use Mashery.

### Tech

Tibco Mashery forwarder uses a number of open source projects to work properly:

* [Lumen](https://lumen.laravel.com/) - Laravel Lumen is a stunningly fast PHP micro-framework for building web applications
* [Guzzle](https://github.com/guzzle/guzzle) - Guzzle is a PHP HTTP client that makes it easy to send HTTP requests and trivial to integrate with web services
